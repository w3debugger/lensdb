Everything you need, nothing you don't

A focused tool that does one thing well, and stays out of your way.

🪶

Native & featherweight

Pure SwiftUI. A single sub-1 MB binary, with no Electron and no bundled runtime. It launches instantly and sips memory.

🐘

PostgreSQL & MySQL

One app, both engines. Flip between them with a click: same browsing, same SQL field, same keyboard shortcuts.

☁️

Local & cloud, together

Open as many connections as you like: local Unix sockets and cloud URLs (Neon, Supabase, RDS, PlanetScale) side by side, grouped into collapsible sections. SSL ready.

🔌

Zero dependencies

It drives the psql / mysql clients you already have. Nothing to link, no drivers to download, nothing to keep updated.

⌘

Browse or query

Click through schemas and tables, or drop into the editable SQL field and run anything with ⌘↩. Paging built in.

✏️

Edit in place

Double-click a cell to change it, or add new rows, then hit Save. LensDB writes the UPDATE / INSERT for you and matches rows by primary key. Nothing is written until you save.

🎯

Faithful results

Exact bigints, decimals, NULLs, and JSON columns. Values round-trip as JSON, never lossily coerced to floats.

🌓

Light & dark

Follows your Mac automatically, or switch to Light or Dark with one click. Sidebar, tables, and results all change instantly.

From zero to browsing in seconds

1

Pick an engine

PostgreSQL or MySQL, from the connection panel.

2

Connect

Local socket, host / port / user, or a pasted connection URL. Add as many as you need.

3

Browse

Databases → tables → rows, grouped by schema with row counts.

4

Query & edit

Tweak the generated SELECT or write your own (⌘↩), and double-click cells to edit. Save writes it back.
